## 引言

React 19 是 React 框架的一次重大更新，引入了多项革命性的改进。本文将从零开始，带你体验 React 19 的核心新特性。

## 项目初始化

使用 Vite 创建 React 19 项目非常简单：

```bash
npm create vite@latest my-app -- --template react-ts
cd my-app
npm install
```

## Server Components

React 19 的 Server Components 允许组件在服务端渲染，减少客户端的 JavaScript 体积：

```tsx
// UserList.tsx - 这是一个 Server Component
async function UserList() {
  const users = await fetch('https://api.example.com/users').then(r => r.json());

  return (
    <ul>
      {users.map(user => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
}
```

## Actions

React 19 引入了 `useActionState` 和 `useFormStatus`，简化了表单处理：

```tsx
import { useActionState } from 'react';

async function submitForm(prevState: any, formData: FormData) {
  const name = formData.get('name');
  // 处理表单数据
  return { success: true, message: `你好，${name}！` };
}

function MyForm() {
  const [state, formAction] = useActionState(submitForm, null);

  return (
    <form action={formAction}>
      <input name="name" placeholder="输入你的名字" />
      <button type="submit">提交</button>
      {state?.message && <p>{state.message}</p>}
    </form>
  );
}
```

## 改进的 Hooks

### use() Hook

新的 `use()` Hook 可以在组件中直接读取 Promise 和 Context：

```tsx
import { use, Suspense } from 'react';

const fetchUser = fetch('/api/user').then(r => r.json());

function UserProfile() {
  const user = use(fetchUser);
  return <div>{user.name}</div>;
}
```

### useOptimistic

乐观更新让用户体验更加流畅：

```tsx
const [optimisticMessages, addOptimisticMessage] = useOptimistic(
  messages,
  (state, newMessage) => [...state, { text: newMessage, sending: true }]
);
```

## TypeScript 集成

React 19 对 TypeScript 的支持更加完善。建议始终使用严格的 TypeScript 配置：

```json
{
  "compilerOptions": {
    "strict": true,
    "jsx": "react-jsx"
  }
}
```

## 总结

React 19 是一个令人兴奋的版本。Server Components 带来了性能革新，Actions 简化了数据流，新的 hooks 让开发体验更加流畅。建议尽快在新项目中尝试这些特性。
